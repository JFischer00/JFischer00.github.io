Projects & Case Studies

These are a few of the personal projects I've completed, which are designed to effectively showcase my data analysis skills.

Cyclistic Case Study

Based on a dataset from Divvy, a Chicago bike sharing company, this case study seeks to answer the question, "What are the most effective ways to market our premium yearly membership to casual riders?"

MLB Hitting Performance

Using a dataset containing player hitting performance by game from 1901-2021, this project seeks to determine whether there is a noticeable trend of overall league improvement over time, and if so, what factors may cause it.

Atlantic Hurricane Strength

Using a dataset containing information on tropical storms and hurricanes from 1851-2015, this project seeks to determine if there is a correlation between storm location and strength.

SQL Practice Queries

This is a collection of SQL queries I completed from the book SQL Practice Problems by Sylvia Moestl Vasilik. The problems are grouped by skill level and range from simple select statements to complex queries involving joins, CTEs, temp tables, and window functions.
